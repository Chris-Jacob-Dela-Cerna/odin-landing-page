# Landing Page

My first project combining HTML and CSS into a complete, styled layout.

🔗 **[Live site](https://chris-jacob-dela-cerna.github.io/landing-page/)**

---

## What I built

A proto-portfolio landing page with four sections: a hero, a projects showcase, a quote block, and a call-to-action. The content is real — my actual projects, my GitHub, my own quote. The structure follows the TOP design spec; the color palette and content are mine.

Built to practice flexbox layouts, CSS nesting, hover states, and linking assets like GIFs and external URLs.

---

## What I was learning

- Flexbox for both page-level layout and component-level alignment
- CSS specificity and when to group selectors
- Responsive-ish wrapping with `flex-wrap`
- The difference between `<button>` and `<a>` for navigation

---

## What it isn't

A finished portfolio. Responsive design for mobile isn't implemented yet. That comes later. This is a foundations exercise that happens to have real content in it.
